## 2026-04-08 - [Sentinel] Fixed Insecure HTTP Link and Missing rel='noopener noreferrer' in Footer\n**Vulnerability:** External WhatsApp link in the footer was using an insecure `http://` protocol instead of `https://` and was missing `target="_blank" rel="noopener noreferrer"` to prevent tabnabbing.\n**Learning:** This codebase had external links with insecure protocols (HTTP) and missing rel attributes, exposing users to MitM and tabnabbing risks.\n**Prevention:** Always use `https://` for external links and include `rel="noopener noreferrer"` when using `target="_blank"`.
